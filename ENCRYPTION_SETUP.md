# خطوات تفعيل نظام التشفير - دليل سريع

## الخطوات المطلوبة:

### 1️⃣ تحديث مفتاح التشفير (ضروري!)

افتح الملف `backend\.env` وغيّر السطر التالي:

```env
ENCRYPTION_KEY=expense-plan-secure-encryption-key-2026-change-me
```

إلى مفتاح سري خاص بك (يفضل أن يكون طويل ومعقد):

```env
ENCRYPTION_KEY=ضع-هنا-مفتاح-سري-طويل-ومعقد-خاص-بك
```

⚠️ **هام جداً:** احتفظ بنسخة من هذا المفتاح في مكان آمن!

---

### 2️⃣ إيقاف خادم Backend

تأكد من إيقاف خادم backend قبل المتابعة:

```powershell
# اضغط Ctrl+C في نافذة الخادم لإيقافه
```

---

### 3️⃣ تفعيل نظام التشفير

شغّل السكريبت التالي الذي سيقوم بكل شيء تلقائياً:

```powershell
.\setup-encryption.ps1
```

هذا السكريبت سيقوم بـ:
- ✓ تحديث بنية قاعدة البيانات (تحويل الأعمدة من NUMERIC إلى TEXT)
- ✓ تشفير جميع البيانات الموجودة
- ✓ التحقق من نجاح العملية
- ✓ عرض تقرير مفصل

---

### 3️⃣ إعادة تشغيل الخادم

```powershell
cd backend
npm start
```

أو استخدم السكريبت المخصص:

```powershell
.\start-backend.ps1
```

---

## ✅ التحقق من التشفير

1. افتح التطبيق وأضف مصروف جديد
2. افتح قاعدة البيانات PostgreSQL وتحقق من الجدول:
   ```sql
   SELECT * FROM monthly_expenses LIMIT 1;
   ```
3. يجب أن ترى حقل `amount` يحتوي على نص مشفر مثل:
   ```
   a3f2b1c4d5e6:8h7g9i0j1k2l3m4n5o6p:9q8r7s6t5u4v3w2x1y0z
   ```

## ⚡ ملاحظات سريعة

✓ جميع المبالغ المالية الآن مشفرة في قاعدة البيانات  
✓ التطبيق يعمل بشكل طبيعي (يفك التشفير تلقائياً)  
✓ البيانات محمية حتى لو تم الوصول المباشر لقاعدة البيانات
